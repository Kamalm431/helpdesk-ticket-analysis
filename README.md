# IT Help Desk Ticket Analysis

## Overview
This project analyzes IT help desk ticket data using Python to identify common issues, resolution times, and operational trends.

## Tools Used
- Python (pandas, matplotlib)

## Dataset
Simulated help desk ticket dataset including issue category, priority, department, and resolution time.

## Key Insights
- Identified most frequent issue categories
- Analyzed average resolution times
- Found patterns in support workload by department

## How to Run
1. Download the dataset (helpdesk_tickets.csv)
2. Run:
   python ticket_analysis.py
